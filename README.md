#### Стояла задача создать прототип сайта по поиску авиабилетов
##### Технологии:
* TS
* React
##### Итог:
* Удалось реализовать базовую валидацию полей(даты,места отправления и прибытия)

### task 2:

##### Технологии:
* TS
* React
##### Итог:
